「gsvexe.x バージョンアップ履歴」-----------------------------------------------

version 0.31

実行後のキーバッファは待たないようにした.

256色の保存時にエラーが出るのをなおした.

保存/復帰スピードを若干アップした.


version 0.30

-l でmint内蔵セーバーみたいにLEDを点滅するようにした.

デフォルト値を変更した.

-k で実行後のキーバッファもクリアするようにした.


version 0.23

新バージョンのミントで動作するように修正した.


version 0.22

-k でキーバッフを捨てるようにした.


version 0.21

６４Ｋスクエアモードでないときはスクロールレジスタをずらさないようにした。

その他若干の修正をした。


version 0.20

大幅な速度改善をした。

グラフィック表示オフの時にバスエラーを出すのを修正。

ハイスピードモードを搭載した。ほぼ、６４Ｋと同等の速度が出せます。ただし、１６色

のみ動作で、コマンド実行前に画面が消える。

-o でコマンド実行前に画面非表示にするようにした。


version 0.10

初期バージョン。
